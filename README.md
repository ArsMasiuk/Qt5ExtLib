### Qt5ExtLib - Legacy Qt Solutions + latest updates for Qt 5.x

An attempt to bring all the legacy Qt Solutions and their recent updates together.

  - Toolbar Dialog
  - MML Widget
  - Pie Menu
  - Thumb Wheel
- Qt Shared Widgets
  - Gradient Editor
  - Find Widget
  - Font Panel
